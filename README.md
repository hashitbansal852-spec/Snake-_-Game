# 🐍 Classic Snake Game

A simple and fun classic Snake game built using vanilla HTML, CSS, and JavaScript. 

🚀 **[Play the Live Game Here!](YOUR_GITHUB_PAGES_LINK_HERE)**

## 🎮 How to Play
* Use the **Arrow Keys** (Up, Down, Left, Right) to steer the snake.
* Press the **Spacebar** to reset or restart the game.
* Eat the food to grow and increase your score without hitting the walls or your own tail!

## ✨ Features
* **Grid-Based Movement:** Built cleanly using standard CSS Grid layout properties.
* **Score & High Score Tracker:** Tracks your current run and saves your best score.
* **Input Lock:** Smooth controls that prevent the snake from accidentally crashing into its own neck.

## 🛠️ Built With
* HTML5
* CSS3 (CSS Grid Layout)
* JavaScript (ES6+)
